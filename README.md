В проекте есть 4 папки:
1) docs - отчет + книжечка с некоторой теорией
2) src - папка с исполяемыми файлами, среди которых:
    - HT1.m - основной файл создания передаточной функции системы
    - draw_pictures.m - код, изначально находившийся в HT1, который был
      позже вынесен в отдельный файл; добавлена ссылка на него в HT1
3) tests - папка, в которой лежит тест, проверяющий систему из HT1 на устойчивость
      и корректность ввода данных
4) utils - основная папка, в которой располагаются управляющие файлы программы:
    - run_tests - файл, удаленно запускающий тесы из папки tests
    - setup u shutdown - файлы открытия и закрытия проекта

Чтобы работать с проектом, необходимо дважды нажать на сам файл проекта,
инициализируя его, после чего выбрать, какое действие произвести:
    - Запустить саму передаточную функцию (файл HT1)
    - Проверить систему (файл run_tests)

Подробное содержание кодов запуска и выключения представлено в отчете.

Shortcut: перенес построение графиков в лругой файл.

Тест: проверяет систему на устойчивость, а также достаточность входных данных
